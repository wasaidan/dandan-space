# 第三空间 · 个人站点

你的个人文字空间。把觉察和记录分享出去的地方。

## 如何添加新文章

打开 `index.html`，找到 `ARTICLES` 数组，在末尾添加一篇新文章：

```javascript
{id:'my-new-article',       // 英文标识，不能重复
 category:'成长觉察 · 2026.07.01',  // 分类/日期
 title:'新文章的标题',              // 标题（留空则不显示）
 content:[
   {type:'p',text:'这是第一段文字。'},
   {type:'p',text:'这是第二段文字。'},
   {type:'line',text:'这是引用块。\n可以换行。'}
 ]}
```

- `type:'p'` = 普通段落
- `type:'line'` = 引用块（左侧有竖线）
- 段落内换行用 `\n`

添加完后保存，刷新网站就能看到。

## 如何部署（让其他人也能访问）

### 方式一：GitHub Pages（推荐，免费）

1. 在 [github.com](https://github.com) 注册账号
2. 创建一个新仓库（Repository），名字随便写，比如 `dandan-space`
3. 把 `third-space` 文件夹里的文件上传到那个仓库
4. 进入仓库 Settings → Pages → 选择 main 分支 → Save
5. 等几分钟，你的网站就会出现在 `https://你的用户名.github.io/dandan-space/`

### 方式二：直接发给别人

把 `index.html` 文件直接发给别人，对方用浏览器打开也能看（但留言功能需要联网）。
